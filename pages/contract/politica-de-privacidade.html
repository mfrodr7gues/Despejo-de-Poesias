Política de privacidade

Última atualização: 2025

1. Dados coletados

O Despejo de Poesias não coleta dados automaticamente (como cookies, rastreadores ou analytics). Os únicos dados coletados são aqueles fornecidos voluntariamente no formulário de contato:

  • Nome;
  • E-mail;
  • Mensagem enviada.

2. Uso das informações

Os dados enviados são utilizados somente para:

  • Responder mensagens enviadas pelo usuário;
  • Entrar em contato, caso necessário.

Nenhuma informação é compartilhada com terceiros.

3. Armazenamento dos dados

Os dados enviados por meio do formulário de contato e das publicações do site são armazenados de forma segura utilizando o Firebase Firestore, um serviço de banco de dados fornecido pelo Google.

O acesso a esses dados é restrito apenas aos administradores autorizados do Despejo de Poesias, por meio de autenticação protegida.

O Firebase adota padrões elevados de segurança e conformidade com a legislação de proteção de dados.

4. Hospedagem

O site Despejo de Poesias é hospedado na plataforma Netlify, que fornece a infraestrutura necessária para a disponibilização do conteúdo na internet.

A Netlify pode coletar dados técnicos básicos, como endereço IP, tipo de navegador e horário de acesso, exclusivamente para fins de segurança, desempenho e operação do serviço.

Esses dados não são utilizados pelo Despejo de Poesias para fins de rastreamento, marketing ou análise de comportamento.

5. Segurança

Adotamos medidas de segurança para proteger os dados armazenados, porém nenhum sistema é 100% seguro.

6. Direitos do usuário

Você pode solicitar:

  • Remoção dos seus dados;
  • Informações sobre os dados armazenados;
  • Correções de dados enviados incorretamente.

7. Alterações na política

Esta política pode ser atualizada a qualquer momento. A versão vigente será sempre a exibida nesta página.

8. Contato

Para dúvidas sobre privacidade, utilize o formulário de contato disponível no site.
